📊 Customer Shopping Behaviour Analysis
🔍 Project Description

This project analyzes customer shopping behavior to identify patterns, trends, and key factors influencing purchasing decisions. By leveraging data analysis tools, the project aims to provide actionable insights that help businesses improve customer engagement, optimize marketing strategies, and increase overall sales.

The dataset includes customer demographics, transaction history, product categories, and purchase frequency. Through data cleaning, exploration, and visualization, meaningful insights are extracted to support data-driven decision-making.

🚀 Objectives
Understand customer purchasing patterns
Identify high-value and repeat customers
Perform customer segmentation
Analyze seasonal and product-based trends
Improve business decision-making using data insights
🛠️ Tools & Technologies
Excel – Data cleaning and preprocessing
SQL – Data querying and extraction
Python (Pandas, Matplotlib, Seaborn) – Data analysis and visualization
Power BI – Interactive dashboards and reporting
📈 Key Analysis Performed
Exploratory Data Analysis (EDA)
Customer Segmentation (e.g., frequent vs occasional buyers)
Sales Trend Analysis (monthly/seasonal trends)
Product Category Performance
Customer Lifetime Value estimation
📊 Dashboard & Visualization

Interactive dashboards were created using Power BI to visualize:

Sales performance
Customer distribution
Purchase frequency
Revenue trends
💡 Key Insights
Identified top-performing product categories
Found patterns in peak shopping periods
